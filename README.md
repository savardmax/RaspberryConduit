# RaspberryConduit
A Pocketmine MP plugin for python Minecraft: Pi edition API socket (MCPI)

Commands
--------

- player.getPos()
- player.setPos(x,y,z)
- player.getDirection()
- chat.post()
- world.getBlock(x,y,z)
- world.getBlocks(x1,y1,z1,x2,y2,z2)
- world.setBlock(x,y,z,id,meta)
- world.setBlocks(x1,y1,z1,x2,x2,z2, id, meta)
- pollBlockHits()
- clearAll()
